PROBLEM STATEMENTCloud security validation is structurally reactive ("violation and patch"). The existing paradigm fails to anticipate vulnerabilities introduced by declarative code complexity.Core Question: To what extent does the topological structure of the Abstract Syntax Tree (AST) in Terraform (HCL) code act as a predictor for the incidence of compliance and security failures in AWS infrastructure?3. PRIMARY OBJECTIVEQuantify "Security Smells" in Infrastructure as Code (IaC). Prove mathematically, via multivariate regression on a local Data Warehouse, that sub-optimal architectural decisions (e.g., monoliths, deep coupling, hardcoded density) inject silenced, measurable vulnerabilities.4.
